A work of interactive fiction.

Red Herrings was created with Inform and has IFID 4F97ECD4-0CB2-11E2-8B68-001CC06F97E5. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Parchment interpreter. You'll need to have Javascript enabled on your web browser.
